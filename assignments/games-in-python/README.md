# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a classic Hangman game in Python using strings, loops, conditionals, and user input. You will practice managing game state, handling guesses, and giving players clear feedback as they work to reveal a hidden word.

## 📝 Tasks

### 🛠️ Word Selection and Game Setup

#### Description
Create the game data needed to start a round of Hangman.

#### Requirements
Completed program should:

- Randomly choose a word from a predefined list.
- Create a hidden version of the word using underscores.
- Set up a limited number of incorrect guesses allowed.
- Show the player the initial game state before guesses begin.

### 🛠️ Guessing Loop and Game Outcome

#### Description
Build the main loop that accepts guesses, updates the hidden word, and ends the game when the player wins or loses.

#### Requirements
Completed program should:

- Prompt the player to enter one letter at a time.
- Reveal correctly guessed letters in the hidden word.
- Track and display the number of incorrect guesses remaining.
- End the game when the full word is guessed or attempts run out.
- Display a clear win or lose message at the end of the game.
